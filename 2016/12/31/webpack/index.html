webpack的基本使用

webpack

webpack 是对前端资源进行【模块化管理工具】兼【打包工具】，它是一个工具，依赖于node，webpack 允许我们把前端代码像 node 代码一样去引入一个包(文件)，把我们写的模块化代码，做语法转化，使其能够在浏览器中运行。

安装webpack

  • npm install -g webpack带-g参数表示全局安装!
  • 通过 webpack 来管理我们的代码，那么我们的代码一般是不会暴露到全局的!

基本使用：合并js模块代码

  • 我们在前端代码像使用 node 代码一样用 module.exports 及 require 来管理(描述)模块的依赖关系, 但是这样的代码是不能直接在浏览器中运行的
  • 我们需要使用 webapck 来管理(转换)我们的代码,使其符合浏览器的规则!
    • 控制台输入 webpack <入口文件名> <输出文件名>
    • webpack 会把我们在代码所依赖的文件(通过 require 获取的文件)合并到一起!
    • 并且【不会产生全局变局】(除非显示的通过 window 点一个属性，因为 window 是一个全局对象)

将css合并到js代码中

1.css-loader 和 style-loader

  • css-loader,把 css 合并到 js 中(没有添加 dom 操作，也就是没把 css 添加到页面中)
  • style-loader,把 css-loader 处理的结果添加到页面(dom)中
    两者配合使用
  • webpack 可以通过 loaders 提供一些特定的功能
  • loader 属性指定具体的 loader ,这个 loader 其实是一个 npm 包
  • css-loader 就是一个 npm 包，项目目录下安装：npm install css-loader --save-dev
  • 同样需要安装 style-loader npm install style-loader --save-dev 可以把 css-loader 写入到 js 的样式添加到 dom 中，通过动态创建了一个 style 标签，通过 innerhtml 把样式代码写入

2.css文件的依赖关系

  • 这样的话，合并的时候只需要把当前的 css 文件合并到 js 文件中即可

3.less-loader

  • npm install less-loader --save-dev
  • 作用: 这个 loader 会读取匹配的 less 文件，然后把 less 文件内容转换为 css 内容
    一般会配合 css-loader 和 style-loader 一起使用!
  • 注意，下载less-loader可能会缺少一个less包：npm install less --save-dev，这个需要另外单独安装!

4.sass-loader

  • sass与less相似 npm install sass-loader --save-dev
  • 这个 loader 会读取匹配的 scss 文件，然后把 less 文件内容转换为 css 内容
  • 注意，下载 sass-loader 可能会缺少一个 node-sass 和 webpack 依赖包需要手动安装 npm install node-sass webpack --save-dev

5.url-loader

  • 作用: 是对 css 中使用的图片进行处理，如果图片比指定的条件小，就转换为 base64 编码形式，如果比指定的条件大就不转换
  • 注意，安装时会缺少一个file-loader包:npm install file-loader --save-dev这个file-loader里会有文件操作!
  • 如果在 css 中有好多小图标,可能有100个，浏览器会再发100个请求!转换成 base64 编码形式可以减少请求服务器的次数
  • name 参数指定如果不生成 base64 时，文件的输出目录及输出的文件
  • [name]表示原文件名,[ext]表示源文件扩展名
  • 如果不指定 name 参数，输出的图片名是随机的
  • limit: 限制转换的大小，值的单位是字节 byte

将ES6语法转换为ES5

babel-loader

  • babel 是专门用来做语法转换的工具，可以进行多种语法转换，但是我们不推荐在浏览器中做语法转换，性能低。可以通过 babel-loader 在 webpack 中使用。
  • npm install bable-loader --save-dev
  • 注意：安装时还需要安装 babel-core:npm install babel-core webpack --save-dev
  • 如果是对 es6 进行语法转换，还需要安装:npm install babel-preset-es2015 --save-dev 包。写代码时，用 es6，在写完之后转换为 es5
