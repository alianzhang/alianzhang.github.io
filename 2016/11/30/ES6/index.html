接触到的ES6知识点总结（未完待续）

ES6

模板字符串

  • 使用``包裹, ${变量} ,最终会输出成变量值
  • 在``之间，可以任意的换行!

const

  • const申明的变量不可改变
  • 如果const申明一个对象,那么这个对的值可以改变，但是不可以让这个对象指向一个新对象!

let

  • let 申明的属性，不管是什么，都是可以变的
  • let 与var申明的变量，主要是作用域上的区别
  • let 的作用域是由【大括号】{}决定,块级作用域
  • 强调,只要与大括号，就会创建一块作用域！
  • var 的作用域是由【函数】决定

find

  • find方法也会遍历数组,参数就是每次遍历的元素
  • 当我们在function里return true时，会把当前遍历的数据作为find的返回值返回！

filter

  • filter方法也会遍历我们的数组
  • 我们在function里return true时，当前元素会被作为数组的一个元素返回
  • 如果return false，当前元素不会返回！
  • 返回值是一个数组

Arrows Function

  • 箭头函数,不会改变this的指向
