Vue

MVVM

  • M: model
  • V: view
  • VM: viewmodel
  • 双向数据绑定！
  • 框架本质就是一个库!
  • 能够
  • 和ag相比，更轻量级，而且不需要创建作用域和

相关指令:

  • v-model
    • 双向数据绑定（页面上可编辑元素比较多的时，可以比较方便获取元素的值，摆脱繁杂的DOM获取）
    • 绑定的标签一般是：input,textarea,checkbox,rang,select
  • v-html
    • <p v-html="msg"></p>
    • 作为元素的属性 不会有闪烁的问题
    • 能够把data的属性值直接渲染到指定的标签中（innerHTMl）
    • 与v-text相比,v-text会把html标签直接(当作普通字符)显示出来
  • v-bind
    • 对标签的属性直接进行赋值
    • v-bind可以简写，即不写

注册事件:

  • v-on:事件名
    • v-on:事件名可以简写为 @事件名
    • <button v-on:click="sayhello">点我吗</button>
    • 简写:
    • 事件指定的方法，在 methods 属性中

v-for

  • v-for=“item in list”
  • 遍历数组的，相当于 ag-repeat=”item in 数组名”（angular）
  • v-if 与 v-else他俩配合使用，所对应的标签要连续写在一起，不显示的元素，所对应的DOM元素被删除
  • v-show 如果为false，页面不显示，但是DOM元素还在。相当于加了display：none
  • ctrl+shift+g 包裹一段代码

nrm

  • 用来切换npm 包的下载地址!
  • 安装nrm: npm install -g nrm
  • 用法: nrm ls // 列出可选的下载地址
    `nrm use taobao` // 设置让npm下载地址变为淘宝地址!
